Задача: лежит ли точка M(xM, yM) внутри треугольника, заданного координатами своих вершин А(xA, yA), В(xB, yB), С(xC, yC) на плоскости?

Входными данными являются координаты вершин A, B, C треугольника и координаты точки М. Определяем принадлежность точки М к плоскости треугольника АВС при помощи метода сравнения площадей.

Суть метода:

1) находим площадь треугольника ABC;

2) находим площади всех треугольников, где одной из вершин является точка М (треугольники ABM, MBC, MAC).

3) находим сумму всех площадей из пункта №2 и сравниваем ее с площадью треугольника ABC.

Вывод: если площадь треугольника ABC равна сумме площадей треугольников ABM, MBC и MAC, то точка M лежит в плоскости треугольника ABC.

Условия необходимые для работы программы:

1.входные данные - натуральные или дробные числа;

2.дробные числа вводятся в виде десятичной дроби, через запятую;

3.не допускается ввод иных символов.
